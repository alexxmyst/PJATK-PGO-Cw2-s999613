# Projekt Biblioteki

```
Projekt MiniBiblioteki. Składa się z klas: 
Książka, 
Czytelnik, 
Biblioteka, 
Main. 
W klasie Main znajduje się scenariusz testowy projektu. 
```
Projekt działający w konsoli, pozwala na śledzenie statusu biblioteki: liczby posiadanych książek i ich dostępności liczby oraz czytelników i ich wypozyczeń. Pozwala równiez na wyszukanie książek po tytule, ich zwrócenie oraz wypożyczenie.
